涟漪/Ripple

输入效果名： 涟漪 或 rpl
参考教程地址 教程作者：冰叹 （bilibili.com）
需要预先安装 Trapcode Particular 粒子插件


示例图片。原图来自《辉夜大小姐想让我告白》漫画111话，用style2paints上色而成。

自动生成一个带有Displacement Map效果的调整层和一个涟漪群合成，并把涟漪群提供给Displacement Map。把想要影响的图层放在调整层下方。涟漪群合成是由Circle特效生成基础的涟漪单元再提供给Particular粒子系统进行量产而成。
